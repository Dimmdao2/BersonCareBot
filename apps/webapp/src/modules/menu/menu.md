# menu

Меню главного экрана по роли пользователя.

Возвращает список пунктов: идентификатор, заголовок, ссылка, статус (доступно / скоро / заблокировано). Для врача — кабинет специалиста и настройки. Для пациента — покупки, скорая помощь, уроки, дневник симптомов, дневник ЛФК, персональный помощник (скоро) и т.д. Используется на странице `/app/patient` для отображения сетки карточек. Конфигурация зашита в сервисе; в перспективе может выноситься в конфиг или БД.
