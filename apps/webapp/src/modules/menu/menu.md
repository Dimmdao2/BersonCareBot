# menu

Меню главного экрана по роли пользователя.

Возвращает список пунктов: идентификатор, заголовок, ссылка, статус (доступно / скоро / заблокировано). Для врача — кабинет специалиста и настройки. Для пациента — разделы контента из БД, запись, кабинет, карточка **`diary`** с подписью **«Статистика»** (`/app/patient/diary`). Используется на странице `/app/patient` для отображения сетки карточек. Конфигурация зашита в сервисе; в перспективе может выноситься в конфиг или БД.
