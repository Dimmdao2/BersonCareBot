# users

Работа с учётными записями пользователей платформы.

Предоставляет получение/создание пользователя по идентификатору и привязке каналов (Telegram, Max и т.д.). Используется при обмене токена и при входе по Telegram initData для создания или обновления записи пользователя и привязок. Детали хранения — в реализациях портов (инфра).
