# media

Медиафайлы: отдача по идентификатору.

Сервис и порт для получения медиа (изображение, видео) по id; реализация порта может отдавать URL или поток из хранилища (в т.ч. мок). Используется при отображении контента с картинками/видео и в API media/[id]. Не зависит от интегратора; вебапп владеет источниками медиа.
