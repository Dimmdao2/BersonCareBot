# media

Медиафайлы: отдача по идентификатору и библиотека CMS.

Сервис и порт для:
- загрузки файлов в библиотеку;
- получения медиа (изображение, видео, аудио, file) по id;
- листинга библиотеки с фильтрацией/сортировкой;
- поиска использований в CMS (`content_pages`);
- hard-delete с подтверждением в UI.

Реализация порта может отдавать URL или поток из хранилища (в т.ч. мок). Не зависит от интегратора; webapp владеет источниками медиа.
