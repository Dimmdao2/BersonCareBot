# patient-home

Данные для главной пациента.

- `repository.ts` — баннер из проекции рассылок (`mailing_topics_webapp`), логи рассылок.
- `newsMotivation.ts` — детерминированная «цитата дня» из `motivational_quotes` (стабильный выбор по UTC-дате и seed пользователя).
- Управление цитатами: `/app/doctor/content/motivation` (роль doctor/admin).
