# app

Корневая папка маршрутов Next.js: страницы и API приложения.

- **app/** — роуты страниц (layout, главная, редиректы).
- **app/app/** — раздел после входа: общая точка входа `/app`, затем ветки по ролям (patient, doctor, settings).
- **app/api/** — обработчики API (авторизация, интегратор, медиа, здоровье и т.д.).

Вся вёрстка и страницы для пользователя и врача собираются из компонентов в `shared/ui` и контента из модулей.
