# lfk

Основной раздел ЛФК на единой странице дневника: `/app/patient/diary?tab=lfk` (legacy `/app/patient/diary/lfk` редиректит сюда через `next.config`). Форма самосоздания комплекса на странице дневника отсутствует; комплексы приходят из назначения врача.

Показывает: описание раздела; форму «Отметить занятие» (выбор комплекса при нескольких, скрытый input при одном) с серверным действием `markLfkSession`; блок «Комплексы»; блок «Статистика» с датами занятий. При отсутствии комплексов — заглушка. Только для пациента. Данные и запись занятия — через `deps.diaries` (listLfkComplexes, listLfkSessions, addLfkSession). Действие в **actions.ts**: проверка доступа, валидация complexId, сохранение сессии, revalidatePath.
