# content

Раздел контента пациента: страницы материалов по идентификатору в адресе.

- **[slug]/** — одна страница материала (`/app/patient/content/[slug]`). Данные из каталога контента по `slug`; показываются заголовок, картинка, текст, блок «Видео» (плейсхолдер). Открывается из разделов «Полезные уроки» и «Скорая помощь». При отсутствии материала — 404.

Каталог и разрешение по slug — в модуле `content-catalog`.
