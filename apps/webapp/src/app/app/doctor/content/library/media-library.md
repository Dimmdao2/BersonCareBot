# media-library

Экран библиотеки файлов в CMS (`/app/doctor/content/library`).

**Точка загрузки для врача:** новые фото/видео/прочие разрешённые типы добавляются здесь (presigned / multipart в MinIO по возможностям клиента). В формах **страниц контента** (обложка, видео), **вставке Markdown** и **карточке упражнения ЛФК** медиа выбирается из уже загруженных файлов через picker (`MediaLibraryPickerDialog` / `MediaLibraryInsertDialog`), а не произвольный URL вручную (кроме сохранённого legacy в контенте и упражнениях — абсолютный `https://`).

Функции:
- просмотр загруженных файлов;
- редактируемое отображаемое название (без изменения исходного имени файла в хранилище);
- фильтрация по типу;
- сортировка по типу, дате загрузки и размеру;
- метаданные в списке: кто загрузил (ссылка на профиль) и разрешение image/video;
- загрузка новых файлов;
- удаление через меню действий (`...`) с предупреждением о местах использования в CMS.
