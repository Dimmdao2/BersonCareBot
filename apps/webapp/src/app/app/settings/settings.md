# settings

Страница «Настройки» (`/app/settings`).

Служит для обратной совместимости старых ссылок и сразу редиректит:

- пациента на `/app/patient/profile`;
- врача/администратора на `/app/doctor`.

Полноценный экран настроек и admin mode — по плану этапа 14, см. `docs/FULL_DEV_PLAN/PLANS/STAGE_14_SETTINGS_ADMIN/FIX_PLAN_STAGE_14.md`.
