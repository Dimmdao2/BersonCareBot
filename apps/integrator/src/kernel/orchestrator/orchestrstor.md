что прислал гейтвэй

Оркестратор управляет сценариями.

Он решает что должно произойти дальше.

Основная задача

Определить последовательность действий на основании:
	•	события
	•	контекста
	•	сценария

Источник сценариев

Сценарии хранятся в:

content/scripts.json

Работа оркестратора

Оркестратор:
	1.	получает событие
	2.	выбирает сценарий
	3.	проверяет условия
	4.	формирует список действий


Что НЕ делает

Оркестратор:
	•	не отправляет сообщения
	•	не работает с API
	•	не читает базу данных
	•	не создаёт задачи

Он только описывает план действий.
