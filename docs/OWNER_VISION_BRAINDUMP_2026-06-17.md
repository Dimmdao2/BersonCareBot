### Плоский добор (мелочь / баги / UX)

1. 🐞 Кнопка «не напоминать в боте» НЕ отключила настройку напоминаний у клиента — **исправить** (проверить).
2. В пуше — «напомнить позже / пропустить» 🆕
3. Завершение упражнения (отметка): опции кол-ва/сложности **убрать из модалки** → постоянный блок с полями; над ним рекомендация-цель специалиста; помнить прошлые значения; правка перед завершением. ◑
4. Клиент добавляет запись в календарь: Google / Yandex / `.ics` в телефон / файлик на почту. 🆕
5. Настройка напоминания после записи: доктор выбирает пресет, клиент правит личный вариант. ◑ *(= Q-R; спец-часть ✅, клиент-часть defer)*
6. Напоминалки: по умолчанию **время**, не интервал; разминки — расписание менять в настройках через БД (доктор). ◑ *(ротация разминок ✅; время-vs-интервал — доработать)*
7. Пациент заливает большие файлы (видео) → серверный рендер в 360/480 + удаление оригинала. ◑
8. Доктор добавляет индивидуальное упражнение в программу ЛФК, заливает в него видео с пациентом (снятое на приёме). 🆕

### Крупные НОВЫЕ направления, которых ещё нет в дорожной карте
- **Модуль «Нутрициология»** (анализы, БАДы, питание-дневник, рецепты) — целый новый вертикальный модуль.
- **ИИ-ассист** (расшифровка заметок, парсинг карт и анализов в историю, разбор анализов, персональные референсы) — новая ИИ-инфраструктура (Claude).
- **Клиническая карта тела + методики** (FM Stecco / Maitland / визуальные боли) — специализированный rehab-инструмент.
- **Биллинг/тарифы/модули + роли-per-org** — монетизация поверх SAAS.
- **Режим клиники + передача пациента** — мультиспециалист (опирается на SAAS).
