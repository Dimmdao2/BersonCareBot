### B1.3 — Завершение упражнения: убрать из модалки

**Что сказал владелец:** «Завершение упражнения — дополнение опций по кол-ву и сложности — убрать из модалки»

**Контекст:** Задача #45 (done, ждёт-приёмки) зафиксировала, что после дрейфа коммита ddb3432e диалог завершения (`ProgramItemCompleteDialog`) вернулся на страницу упражнения пациента. В #45 задан ВОПРОС ВЛАДЕЛЬЦУ: диалог — намеренный UX или регресс QW-A1?

**Неясность:** Фраза «убрать из модалки» может означать:
- (а) убрать поля qty/difficulty из существующего диалога (диалог остаётся, упрощается)
- (б) убрать диалог полностью (вернуть inline-форму, как в QW-A1)

**Рекомендация:** Ответ на B1.3 является одновременно ответом на вопрос #45. Нужна одна директива владельца: «диалог с упрощённым подтверждением» (вариант а) или «только inline без диалога» (вариант б). После ответа #189 и закрытие #45 объединить.
