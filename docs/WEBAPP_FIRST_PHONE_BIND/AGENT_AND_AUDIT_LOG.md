---

## Шаблон записи (агент / человек)

```
### YYYY-MM-DD — <краткий заголовок>
- **Этап:** STAGE_XX_...
- **Действия:** …
- **Артефакты:** PR / коммит / ветка
- **CI:** pass / fail
```

## Шаблон записи (аудит)

```
### AUDIT YYYY-MM-DD — <область>
- **Этап / scope:** …
- **Проверено по:** STAGE_XX чек-лист (пункты: …)
- **Найдено:** …
- **Статус:** OK / gaps (список) / blocked
- **Follow-up:** … (только если есть)
```
