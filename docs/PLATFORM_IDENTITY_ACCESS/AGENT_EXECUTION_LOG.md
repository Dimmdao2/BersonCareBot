## Единый гейт контакта (бот + Mini App) — выполнение плана

**Модель двух слоёв:** (1) **основной контроль в боте** — при отсутствии привязанного номера любое релевантное действие в чате ведёт в запрос контакта, а не в полноценное меню/WebApp; (2) **страховка в Mini App** — если пользователь уже открыл WebApp без tier **patient**, оверлей «Предоставить контакт» + M2M пуш клавиатуры в чат + закрытие WebView. Дополнительно: без SMS в миниаппе при привязке через мессенджер; `ctx=bot` и cookie-fallback детекта миниаппа; человекочитаемые ошибки записи при `patient_activation_required`.
