## F. Расписание — недельная настройка прямо в календарной странице
> Скоуп крупнее quick-win — разрешено дробить на подзадачи. Точка входа: `doctor/schedule/tabs/ScheduleWorkTab.tsx`
> (рабочие дни/время: `WD_BASE=/api/admin/booking-engine/working-days`, шаблоны
> `TPL_BASE=/api/admin/booking-engine/working-schedule-templates`) + `settings/BookingSoloScheduleSection.tsx`.
> Сейчас врач выделяет КОНКРЕТНЫЕ даты в календаре и справа настраивает им время. Недельный режим — НЕ отдельная
> вкладка/режим, а часть того же календаря.
>
> **ГЛАВНОЕ — единое пространство.** Это ОДИН общий календарь, на котором сразу видно всё разом: и недельный шаблон
> (по дню недели), и точечные правки конкретных дат, и выходные. Пример того, что врач видит на одном экране:
> все понедельники 11–18, два конкретных вторника 10–16, пятница — выходной. Никакого переключения «недельный
> режим / календарный режим»: врач просто кликает либо по заголовку дня недели (правит весь столбец = шаблон),
> либо по конкретным датам (правит/чистит точечно), а календарь всегда отображает фактический итог по каждой дате.

11. **Недельная настройка по дню недели (столбцом).** В календарной настройке расписания дать клик по ячейке
    **заголовка дня недели** («Пн», «Вт», …) → выделяется **столбцом весь день недели** (клик на «Пн» — все понедельники,
    «Вт» — все вторники и т.д.). Для выделенного диапазона справа открывается **та же панель настройки времени**, что
    сейчас для выбранных календарных дней. Отличие: для недельного режима по умолчанию включена **новая галочка
    «Постоянное расписание»** — значит «в этот день недели я работаю так во все месяцы» (сохранять как недельный
    шаблон `working-schedule-templates`, а не как набор дат).
    **DoD:** клик по «Пн» выделяет все понедельники; справа панель времени; галочка «постоянное расписание» по умолчанию
    включена; сохранение → недельный шаблон применяется ко всем месяцам; **на том же календаре каждая дата показывает
    своё фактическое время (от шаблона), точечные правки и выходные различимы визуально**; скриншот.

12. **Точечная отмена расписания у выбранных дат (выходные-override).** После недельной настройки можно отдельно
    выделить конкретные дни в календаре и **очистить у них расписание** (сделать выходными). UX как сейчас для
    календарной настройки: выделил даты → справа кнопка **«Очистить»** → день стал выходным. Это override на конкретную
    дату поверх недельного шаблона; остальные понедельники (или др. дни недели) продолжают читать общее недельное
    расписание. Не путать с правкой шаблона — чистится только выбранная дата.
    **DoD:** выделение дат + «Очистить» делает их выходными, недельный шаблон не меняется, прочие дни той же недели
    остаются рабочими по шаблону; скриншот.
