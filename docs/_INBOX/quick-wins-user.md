## A. Упражнение / программа — всё на СТРАНИЦЕ элемента, не в модалках

1. **Отметка выполнения — inline на странице (убрать модалку).** Поля повторы/подходы/вес — постоянным блоком
   на странице упражнения; над полями — рекомендация-цель врача; подставлять прошлые значения, дать поправить перед
   завершением. Модалку `diary/QuickAddPopup.tsx` убрать с этого пути.
   **DoD:** на странице упражнения виден блок полей + цель + прошлые значения; завершение без модалки; скриншот.
2. **Список → иконки + клик открывает страницу.** Убрать из плитки кнопку «пометить выполненным» («отправить видео»
   уже убрана — подтвердить). Клик по элементу → страница упражнения (не модалка). Снизу плитки — иконки:
   есть комментарии / есть новые / противопоказания / зелёные точки.
   **DoD:** в плитке нет кнопок-действий, есть иконки; клик ведёт на страницу; скриншот.
3. **Комментарии врача — на странице упражнения.** Читать и добавлять комментарии прямо на странице элемента (не в модалке).

4. **🐞 Зелёные точки за день.** Должно показывать несколько точек, если упражнение отмечено несколько раз за день.
