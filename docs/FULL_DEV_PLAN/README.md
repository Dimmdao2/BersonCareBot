# BersonCare — Полный план развития

> Дата создания: 2026-03-22

## Содержание папки

| Файл | Назначение |
|------|-----------|
| **RAW_PLAN.md** | Исходный текст требований от владельца продукта (без изменений) |
| **ANALYSIS.md** | Результаты анализа: блоки задач, приоритеты, сложность, библиотеки, пропуски, противоречия, риски |
| **ROADMAP.md** | Дорожная карта: 20 этапов, подэтапы, стек, паттерны, тесты |
| **PLANS/** | Папка с детальными инструкциями по каждому этапу (14 подпапок по блокам) |

## Как читать

1. Начните с **RAW_PLAN.md** для понимания общего замысла.
2. Прочитайте **ANALYSIS.md** для оценки рисков, пропусков и рекомендаций.
3. Используйте **ROADMAP.md** как навигационную карту по этапам.
4. Передавайте файлы из **PLANS/** агентам для выполнения конкретных задач.

## Статистика

- **20 этапов**, ~90 подэтапов
- **14 функциональных блоков**
- **~25–45 миграций БД**
- **~70–110 новых компонентов**
