## Уточнение владельца (2026-06-23, второе сообщение)
1. **Старую страницу Clients удаляем** — чистим файлы старого `/app/doctor/clients/*` (UI).
   История сохранена в коммитах GitHub, бояться нечего. Делать сейчас.
2. **Чат из заявок (leads) НЕ ведём в карточку пациента.** Заявка — это ещё не клиент, а только
   потенциальный (человек с заявкой). Поэтому открывать карточку пациента из заявки неправильно.
3. **Решение:** открывать чат из заявки как **модалку с чатом** (без карточки пациента).
4. Нам в любом случае нужна **переиспользуемая, аккуратная модалка с чатом** для разных страниц —
   делаем её один раз и из заявок открываем её же.
