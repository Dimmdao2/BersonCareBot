## B1. Панч-лист — напоминания/клиент/визиты

1. Кнопка «не напоминать в боте» не отключила настройку напоминаний у клиента — ИСПРАВИТЬ!
2. В пуше — «напомнить позже» или «пропустить»
3. Завершение упражнения — дополнение опций по кол-ву и сложности — убрать из модалки
4. Клиенту: запись добавлять в календарь (Google / Яндекс / файлик в календарь телефона; в идеале на почту слать файлик — их почта понимает) [DUP: #81]
5. Настройка напоминания после записи — доктор выбирает пресет, клиент выбирает личный вариант (правит пресет)
6. Напоминалки — лучше не интервальные по умолчанию, а время. И на разминки — расписание менять в настройках через БД (доктор настраивает)
7. Пациент заливает большие файлы (видео упражнений) — на сервере сразу рендер в мини-размер (360/480) + удаление оригинала
8. Доктор может в программу ЛФК добавить индив. упражнение, залить в него видео с пациентом — личное упражнение на приёме
