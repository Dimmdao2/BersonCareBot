# Фаза 1 — Лендинг на `/`

**Цель:** маркетинговая обложка на корне сайта без входа в кабинет и без авто‑редиректа в `/app`. Подготовка места под блок установки (логика кнопки — [фаза 2](PHASE_02_INSTALL_FLOW.md)).

**Пререквизиты:** принципы и scope — [фаза 0](PHASE_00_PRINCIPLES_AND_SCOPE.md).

## Scope (код)

- `apps/webapp/src/app/page.tsx` — лендинг на `/` **без** `redirect("/app")`.
- Вынесенные компоненты: `src/shared/ui/marketing/` (лендинг + слот установки из фазы 2).
- Статика в `public/` — только если нужны ассеты лендинга (отдельные ассеты лендинга не вводились).
