## Зачем
Сейчас стоит **блок: вход только как «приложение» (standalone/PWA)** — нельзя зайти как на обычный сайт.
Многим удобно пользоваться **с сайта на компе**, поэтому нужно: открывать и как сайт, и иметь нормальный
**десктоп-вид**, и при этом везде уметь **ставиться как приложение**.

## Что сделать (интент владельца)
1. **Снять блок «только приложение»** → приложение открывается **и как обычный сайт**.
2. В режиме сайта (не установлен) — **настойчиво показывать сверху баннер «установить как приложение»**
   (видный, постоянный; в app-режиме — не показывать).
3. **Нормальная десктоп-вёрстка** (адаптив для большого экрана; многие смотрят с компа).
4. **Установка как приложение на десктоп-Chrome** (PWA install / обёртка).
5. **Mac** — тоже дать способ поставить как приложение.
6. **Проковырять ВСЕ вёрстки, чётко все варианты:** mobile-web · mobile-installed-PWA · desktop-web ·
   desktop-installed-PWA · Mac · виды специалиста и клиента отдельно.
