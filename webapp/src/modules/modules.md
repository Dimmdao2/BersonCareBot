# modules

Доменные модули вебаппа: бизнес-логика без привязки к фреймворку.

Каждый модуль отвечает за свою область: авторизация, пользователи, меню, уроки, скорая помощь, кабинеты пациента и врача, покупки, дневники (симптомы, ЛФК), каталог контента, настройки каналов, медиа, интегратор (события от бота, напоминания). Хранение данных делегировано портам; реализации портов лежат в `infra/repos`. Страницы и API вызывают сервисы модулей через объект, собранный в `app-layer/di`.
