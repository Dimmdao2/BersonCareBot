# doctor

Главная страница врача/админа (`/app/doctor`).

Показывается только пользователям с ролью врач или админ. Содержит сообщение из «рабочего пространства» и блок «Список пациентов» (сейчас заглушка или список из конфигурации). Данные из `doctorCabinet.getDoctorWorkspaceState()`. Корневая страница раздела врача, без кнопки «Назад».
