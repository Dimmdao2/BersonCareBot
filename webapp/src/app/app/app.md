# app (внутренний)

Раздел приложения после входа: общая точка входа и ветки по ролям.

- **page.tsx** — страница `/app`: проверка сессии, редирект пациента в `/app/patient`, врача/админа в `/app/doctor`; для неавторизованных — экран входа и блок авторизации.
- **patient/** — все страницы пациента (главное меню, дневники, уроки, кабинет, настройки и т.д.).
- **doctor/** — главная страница врача/админа.
- **settings/** — общие настройки (профиль, каналы, уведомления); доступны и пациенту, и врачу.

Условие показа: пользователь авторизован; выбор ветки по роли из сессии.
