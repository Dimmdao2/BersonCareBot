# symptoms

Страница «Дневник симптомов» (`/app/patient/diary/symptoms`).

Показывает: пояснение, что записи добавляются в боте; блок «Отслеживаемые симптомы» (названия); блок «Статистика» — список записей с баллом 0–10, типом (в моменте / за день) и датой. При отсутствии данных — заглушка. Только для пациента, кнопка «Назад» — в главное меню пациента. Данные из `deps.diaries.listSymptomTrackings` и `listSymptomEntries`.
