# cabinet

Страница «Кабинет клиента» (`/app/patient/cabinet`).

Показывает краткое описание кабинета (причина обращения и т.п.), метку ближайшей записи и блок «Ближайшие записи» — список предстоящих приёмов со ссылками. При отсутствии записей — заглушка. Только для пациента. Данные из `patientCabinet.getPatientCabinetState` и `getUpcomingAppointments`.
