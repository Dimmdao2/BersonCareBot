# ui

Компоненты интерфейса, общие для страниц приложения.

- **AppShell** — оболочка страницы: контейнер, верхняя панель (заголовок, опционально кнопка «Назад», имя/роль пользователя, ссылка «Настройки»), область контента. Используется на всех страницах после входа.
- **AuthBootstrap** — блок входа: при наличии токена в URL — обмен на сессию и редирект; при отсутствии — попытка входа по initData Telegram Mini App. Показывается на `/app` неавторизованному пользователю.
- **FeatureCard** — карточка раздела: заголовок, описание, статус (доступно / скоро / заблокировано), опционально ссылка. Компактный вариант — только заголовок. Используется на главной пациента для пунктов меню.

Внешний вид задаётся классами из `app/globals.css`.
